# timer 计时模块

## 1 createTimeout

创建一个超时计时器，支持中途重置、超时执行自定义逻辑

### 1.1 示例：超时报错

```typescript
const { resolve, reset, promise, isTimeout } = createTimeout(50)
isTimeout.value // false
reset(5)
try {
  await promise
} catch (e) {
  e // Error: timeout!
}
```

### 1.1 示例：超时执行自定义逻辑

```typescript
const { resolve, reset, promise, isTimeout } = createTimeout(50, () => {
  console.log('timeout')
})
await promise // print 'timeout'
```
